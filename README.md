# EN
## How to install
Drop Gwkhtmsat.dll in to EXILED\Plugins.
## Config example
```
GT:
  is_enabled: true
  lines: [thing 1, thingy thing, text, yes]
```
Result will be:
 * random unit-random numbers
 * thing 1
 * thingy thing
 * text
 * yes
 * ...
## PRO-TIP
Text in top right will be shown only to NTF units
